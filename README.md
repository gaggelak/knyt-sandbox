# knyt-sandbox

Sandbox repository for [KNYT](https://github.com/gaggelak/knyt.ai) agents.

When KNYT spawns a coding agent, the agent clones this repo, creates a branch, makes changes, and opens a PR. Treat all branches and PRs here as agent-generated and disposable.

## Structure

This repo starts intentionally bare. Agents will populate it as they work.
